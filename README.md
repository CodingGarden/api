# Patreon Pledges API

A simple API to list the active pledges for a given patreon campaign.

## Endpoints

`GET /pledges`

## Setup

```sh
npm install
cp .env.sample .env # update with your patreon access token and campaign id
```

## Lint

```sh
npm run lint
```

## Development

```sh
npm run dev
```

## Package Audit with snyk

```sh
npm run audit
```
